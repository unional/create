# @unional/devpkg-node

[![NPM version][npm-image]][npm-url]
[![NPM downloads][downloads-image]][downloads-url]

Initializes and manages simple repository.

## Install

Use `@unional/create` to install this package.

```sh
npm init @unional <repository-name>
```
